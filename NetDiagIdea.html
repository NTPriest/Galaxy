SOHO #1


Proof-of-Concept (PoC) Visual:



Physical Diagram:


Description:

This is a small office/home office (SOHO) lab network built with proper segmentation in mind. It includes VLAN-tagged trunk uplinks, port security per access switch, no native VLANs for better security, and a firewall running pfSense - with opn_vpn. Devices like IoT, CCTV, and consoles are isolated appropriately to reduce risk and improve control.


Key Concepts:

  • VLANs: Properly tagged traffic separation.
  • Trunk & Access port logic (802.1Q).
  • No native VLAN to prevent leaks.
  • Basic Port Security (per-port max device count).
  • pfSense NIC directly on choke point.
