About Sanctu_Net

Sanctu_Net is a long-term, privacy-oriented mesh homelab project designed around digital sovereignty, decentralized routing, and network self-healing. It aims to be a testbed for sovereign networking ideas — running entirely on open protocols and manual configuration, for those who want full-stack control.

Core Ideas:

  • Mesh-style Routing: Nodes can dynamically act as clients or relays ("Czars") in a decentralized overlay.
  • CarouselSW: Rotating routing logic that refreshes routes periodically to resist profiling.
  • Blockchain-based Auth: Lightweight, file-based identity layer for node authentication.
  • Tor/ICMP Failover: Optional hidden service routing fallback when WAN is unstable.

Each component is tested in a VirtualBox LAN environment first, using a mix of Bash, Lua, and OpenVPN/WireGuard scripts. The end goal is a WAN-ready sovereign networking setup with full node control and no external dependencies.
